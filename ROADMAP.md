# ccmux Roadmap

## Features

- [ ] **Telegram chat support** — make ccmux work across all 3 scenarios:
  - [ ] Direct Messages (DM / private chat)
  - [ ] Classic group chat
  - [ ] Group with topics (forum-style supergroup)

- [ ] **Installer: upgrade only on version change** — skip reinstall if the installed version matches the latest

- [ ] **Onboarding** — smooth first-run experience:
  - [ ] Interactive config creation (bot token, chat IDs, etc.)
  - [ ] System service installation (systemd / launchd)
